6.1.13.7. LruCache¶

This module implements a Least Recently Used cache using a bidirectional linked list.

class Babel.Tools.LruCache.CacheElement(cache_manager, obj, acquire=False)[source]¶

Bases: object

This class implements a cache element of the LRU cache.

The stored object must implement the Object Protocol defined by the abstract class ObjectProtocol.

The reference counter is set to one if the acquire flag is set.
